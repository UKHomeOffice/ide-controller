Each package has it's own package.json, node_modules, .eslintrc.js etc. These packages should be as independent as possible from the root package. It is possible to use packages from the root but these should only be devdependencies.

Lerna is used to manage these packages. It's a tool that optimises the workflow around managing multi-package repositories with git and npm and is used by projects like Babel, React, Angular, Ember, Meteor, Jest.
